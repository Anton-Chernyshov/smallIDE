# smallIDE
<h1>ABOUT</h1>
<p>smallIDE is a text editor built entirely in python.</p>
<h1>INSTALLATION</h1>
<p>Prior to installation, ensure you have <a href="https://www.python.org/downloads/">python</a> installed. This only uses the stdlibs inbuilt into python </p>
